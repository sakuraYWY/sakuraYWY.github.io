文本格式化标签

b 加粗 strong 加粗
u 下划线 ins 下划线

图片标签
<img src=” “ alt=””>
需要进行标签属性的设置src=””,alt=””.src是属性名，引号中是属性值.

属性注意点：

  1. 标签的属性写在开始标签内部
  2. 标签上可以同时存在多个属性
  3. 属性之间以空格隔开
  4. 标签名和属性之间以空格隔开
  5. 属性之间无顺序

属性名：
alt：替换文本
若图片加载不出来时，以文字代替。

title：提示文本
当鼠标悬停在图片上时显示的文本，不仅可以用于图片标签，还可以用于其他标签。

width和height(一般只设置一个)
设置图片的高度和宽度，若只设置了其中一个，另一个就会等比例缩放（这样图片不会变形）

路径介绍

  1. 绝对路径：目录下的绝对位置，通常从盘符开始。
    例如：C:\Users\杨宛荧\Desktop\3.png
    完整的网络地址 https://sakuraywy.github.io/about/#lg=1&slide=0

linux系统：\为根目录

  1. 相对路径：从当前文件开始找目标文件
    分类：
    同级目录(图片在同一个文件夹中)
    <\img src=”目标图片.jpg”>
    <\img src=”./目标图片.jpg”>
    下级目录（图片所在文件夹与当前在同一个文件夹中）
    <\img src=”img/3.png”>
    上级目录（当前所在的文件夹与图片在同一个文件夹中）
    返回上一级../
    <\img src=”../3.png”>

    注：<后的\去掉，因为格式问题加上 的

音频标签
<\audio src=”./music.mp3” controls>

属性名 说明
src 音频的路径
controls 显示播放的控件
autoplay 自动播放(部分浏览器不支持)
loop 循环播放

注意：目前只支持三种格式：MP3，wav，ogg

视频标签
<\video src=”./video.mp4” controls>

属性名 说明
src 视频的路径
controls 显示播放的控件
autoplay 自动播放(谷歌浏览器需要增加 muted实现静音播放)
loop 循环播放

链接标签
<\a href=”./目标网页.html”>超链接
注意：

  1. 当开发网站初期，还不知道跳转链接时，href的值可以写#（空链接）
  2. href的标签值是跳转地址

属性名：target
属性值：网页的打开形式

取值 效果
_self 默认值，在当前窗口中跳转（覆盖原网页）
_blank 在新窗口中跳转（保留原网页）

例如：<\a href=”https://sakuraywy.github.io/" target=”_blank”>我的博客网址</a>

html基础

列表标签
  1. 无序列表
    标签名 说明
    ul 表示无序列表的整体，用于包囊标签
    li 表示无序列表的每一项，用于包含每一项的内容

特点：列表的每一项前默认显示圆点标识
ul只能包含li标签，li标签可以包含任意内容
2. 有序列表

标签名 说明
ol 表示无序列表的整体，用于包囊标签
li 表示无序列表的每一项，用于包含每一项的内容

特点：列表的每一项前默认序号
ol只能包含li标签，li标签可以包含任意内容

  1. 自定义列表
    标签名 说明
    dl 表示自定义列表的整体，用于包裹dt/dd标签
    dt 表示自定义列表的主题
    dd 表示自定义列表的针对主题的每一项内容

注意：dd前会默认显示缩进效果；
dl标签中只允许包含dt/dd标签

表格标签

标签 说明
table 表格整体，用于包含多个tr
tr 表格每行，可用于包含td
td 表格单元格，可用于包含内容

属性名 属性值 效果

步骤：

  1. 明确合并哪几个单元格
  2. 通过左上原则，确定保留谁删除谁
    上下合并保留上，左右合并保留左
  3. 给保留的单元格设置格式
    属性名 属性值 说明
    rowspan 合并单元格个数 跨行合并（上下合并）
    colspan 合并单元格个数 跨列合并（左右合并）

注意：只有同一个结构标签中的单元格才能合并，不能跨结构标签合并（不能跨thead、tbody、tfoot合并）

CSS

认识
  1. 定义：层叠样式表
  2. 作用：给页面中的HTML标签设置样式
  3. 写在style标签中，style标签一般写在head标签里面，title标签下面
  4. 选择器{css属性}，写完加分号。选择器的作用是查找标签。
  5. 注释是/**/,快捷键是ctrl+/
CSS的引入方式
  1. 内嵌式：写在style标签内.小案例使用
  2. 外联式：写在一个单独的.css文件中。项目中使用
  • 需要通过link标签在网页中引入

    <\link rel=”stylesheet” href=”./mycss.css”>

  1. 行内式：CSS写在标签的style属性中。配合js使用
  • 现在不推荐使用，之后配合js使用
选择器

标签选择器
通过标签名，找到网页中所有这类标签，设置样式
注意：

  1. 选择的是一类标签，而不是单独某一个
  2. 无论嵌套关系有多深，都能找到对应的标签 类选择器
  3. 结构： .类名（css属性名：属性值；）
  4. 作用：通过类名，找到网页中所有带有这个类名的标签，设置样式
    注意：
  • 所有标签都有class属性，属性值为 类名
  • 类名可由数字、字母、下划线、中划线组成，但不能以数字或中划线开头
  • 一个标签可以有多个类名，不同类名空格隔开
  • 类名可以重复，一个类选择器可以同时选中多个标签

id选择器
结构：#id属性值{css属性名：属性值；;}
作用：通过id属性值，找到页面中带有这个id属性值的标签，设置样式
注意：

  • 所有标签都有id属性
  • id属性类似于身份证号码，在一个页面中是唯一的，不可重复
  • 一个标签上只能有一个id属性值
  • 一个id选择器只能选中一个标签
字体

字体大小
属性名：font—size
取值：数字+px
注意：

  • 谷歌浏览器默认文字大小是16px
  • 单位需要设置，否则无效

字体粗细
属性名：font-weight
取值：
1.关键字：正常：normal
加粗：bold
2.纯数字：100——900的整数
正常：400
加粗：700
注意：

  • 不是所有字体都提供了九种粗细
  • 实际开发以正常和加粗两种使用最多

文字倾斜
属性名：font-style
取值：正常：normal
倾斜：italic

  或者<em></em>

字体
属性名：font-family
常见取值：具体字体1，具体字体2……

  • 具体字体：“Microsoft YaHei”、微软雅黑（windows默认）、黑体、宋体
  • 字体系列：sans-serif、monospace等
    渲染规则：
  1. 从左到右按照顺序查找，如果电脑中未安装该字体，则显示
  2. 如果都不支持，会根据操作系统，显示最后字体系列
    注意：
  3. 若字体名称中存在多个单词，推荐用引号包裹
  4. 最后一项字体系列不需要用引号包裹
  5. 网页开发时尽量使用系统常见自带字体

常见字体系列

  1. 无衬线字体（sans-serif）
    特点：文字壁画粗细均匀，首尾无修饰
    场景：网页中采用
    常见该系列字体：黑体、Arial

  2. 衬线字体（serif）
    特点：文字笔画粗细不均，首尾有笔锋修饰
    场景：书籍
    常见该系列字体：宋体

  3. 等宽字体
    特点：每个字母或文字的宽度相等
    场景：一般用于程序代码编写，有利于代码的阅读和编写
    常见该系列字体：Consolas、fira code

字体font相关属性的连写
属性名：font
取值：style weight size family
省略要求：只能省略前两个，如果省略了相当于设置了默认值
注意点：
font:style weight size 字体

  • 如果需要同时设置单独和连写形式
  • 要么把单独的样式写在连写的下面
  • 要么把单独的样式写在连写的里面

文本样式

  1. 文本缩进：text-indent 一般为text-indent:32px或text-indent:2em;
  2. 文本水平对齐方式：text-alight
    属性值：left 左对齐
    right 右对齐
    center 居中
  • 如果要让文本水平居中，text-align属性给文本所在标签（文本的父元素）设置
  • 居中的元素可以有：
    1. 文本

  1. span标签、a标签

  2. input标签、img标签

  3. 文本修饰：text-decoration
    属性值：underline 下划线
    none 无装饰线

  • 开发中常用text-decoration：none;来清除a标签默认的下划线
  1. 行高：控制一行的上下行间距
    属性名：line-height
    取值：
  • 数字+px
  • 倍数（当前标签font-size的倍数

    应用：

  1. 让单行文本垂直居中可以设置line-height：文字父元素高度

  2. 网页精准布局时，会设置line-height：1 可以取消上下间距

  3. 如果同时设置了行高和font连写，注意覆盖问题
    font：style weight size/line-height family

  4. 标签水平居中方法总结margin： 0 auto
    注意：

  • 如果要让div、p、h（大盒子）水平居中，直接给当前元素设置即可
  • margin： 0 auto 一般针对于固定宽度的盒子，如果大盒子没有设置宽度，此时默认沾满父元素的宽度
背景

背景颜色
属性名：background-color（bgc）
属性值：关键字、rgb表示法、rgba表示法、十六进制（#开头）……
注意：

  • 背景色默认是透明：rgba（0，0，0，0）、transparent
  • 背景颜色不会影响盒子大侠，并且还能看清盒子的大小位置，一般在布局中会习惯先给盒子设背景色。

背景图片
属性名：background-image(bgi)
属性值：background-image：url(‘图片路径’)
注意：

  • 背景图片中url中可以省略引号
  • 背景图片默认是水平和垂直方向平铺的
  • 背景图片仅仅是值给盒子内起到装饰的作用，类似于背景颜色，不能撑开盒子

背景平铺
属性名：background-repeat(bgr)
属性值：repeat 默认的水平和垂直方向都平铺
no-repeat 不平铺
repeat-x 沿水平方向平铺
repeat-y 沿垂直方向平铺

背景位置
属性名：background-position（bgp）
属性值：水平方向位置 垂直方向位置

  1. 方位名词（最多表示9个位置）
  • 水平方向：left、center、right
  • 垂直方向：top、center、 bottom
  1. 数字+px（坐标）
  • 原点（0.0）在盒子的左上角
    x轴水平向右
    y轴垂直向下
    操作：将图片左上角于坐标点重合即可

注意：两种可以混合使用

背景相关属性连写
属性名：background（bg)
推荐：background：color image repeat position
注意：

  • 可以按照需求省略
  • 特殊情况：在pc端，若盒子大小和背景图片大小一样，可以直接写background: url()

导航制作

布局顺序：

  1. 从外到内，从上到下
    每一个盒子的样式
  2. 宽高
  3. 辅助的背景颜色
  4. 盒子的模型部分：boder、padding、margin
  5. 其他样式：color、font—、text……

浮动

行内块
display：inline-block;
若代码换行，会产生空格间距

作用：网页布局
特点：脱离标准流，可以覆盖标准流里的元素，下一个浮动会浮动在上一个浮动左右，浮动标签是顶对齐的
margin-top：50px可以使之下来

css书写顺序

  1. 浮动/display
  2. 盒子模型：margin、border、padding、宽度、高度、背景色
  3. 文字样式
