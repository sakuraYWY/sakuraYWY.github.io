图片标签

<img src=” “ alt=””>
需要进行标签属性的设置src=””,alt=””.src是属性名，引号中是属性值.

属性注意点：

  1. 标签的属性写在开始标签内部
  2. 标签上可以同时存在多个属性
  3. 属性之间以空格隔开
  4. 标签名和属性之间以空格隔开
  5. 属性之间无顺序

属性名：
alt：替换文本
若图片加载不出来时，以文字代替。

title：提示文本
当鼠标悬停在图片上时显示的文本，不仅可以用于图片标签，还可以用于其他标签。

width和height(一般只设置一个)
设置图片的高度和宽度，若只设置了其中一个，另一个就会等比例缩放（这样图片不会变形）

路径介绍

  1. 绝对路径：目录下的绝对位置，通常从盘符开始。
    例如：C:\Users\杨宛荧\Desktop\3.png
    完整的网络地址 https://sakuraywy.github.io/about/#lg=1&slide=0

linux系统：\为根目录

  1. 相对路径：从当前文件开始找目标文件
    分类：
    同级目录(图片在同一个文件夹中)
    <\img src=”目标图片.jpg”>
    <\img src=”./目标图片.jpg”>
    下级目录（图片所在文件夹与当前在同一个文件夹中）
    <\img src=”img/3.png”>
    上级目录（当前所在的文件夹与图片在同一个文件夹中）
    返回上一级../
    <\img src=”../3.png”>

    注：<后的\去掉，因为格式问题加上 的

音频标签
<\audio src=”./music.mp3” controls>

属性名 说明
src 音频的路径
controls 显示播放的控件
autoplay 自动播放(部分浏览器不支持)
loop 循环播放

注意：目前只支持三种格式：MP3，wav，ogg

视频标签
<\video src=”./video.mp4” controls>

属性名 说明
src 视频的路径
controls 显示播放的控件
autoplay 自动播放(谷歌浏览器需要增加 muted实现静音播放)
loop 循环播放

链接标签
<\a href=”./目标网页.html”>超链接
注意：

  1. 当开发网站初期，还不知道跳转链接时，href的值可以写#（空链接）
  2. href的标签值是跳转地址

属性名：target
属性值：网页的打开形式

取值 效果
_self 默认值，在当前窗口中跳转（覆盖原网页）
_blank 在新窗口中跳转（保留原网页）

例如：<\a href=”https://sakuraywy.github.io/" target=”_blank”>我的博客网址</a>

列表标签

  1. 无序列表
    标签名 说明
    ul 表示无序列表的整体，用于包囊标签
    li 表示无序列表的每一项，用于包含每一项的内容

特点：列表的每一项前默认显示圆点标识
ul只能包含li标签，li标签可以包含任意内容

去掉列表的符号
li{
list-style: none;
}
2. 有序列表

标签名 说明
ol 表示无序列表的整体，用于包囊标签
li 表示无序列表的每一项，用于包含每一项的内容

特点：列表的每一项前默认序号
ol只能包含li标签，li标签可以包含任意内容

  1. 自定义列表
    标签名 说明
    dl 表示自定义列表的整体，用于包裹dt/dd标签
    dt 表示自定义列表的主题
    dd 表示自定义列表的针对主题的每一项内容

注意：dd前会默认显示缩进效果；
dl标签中只允许包含dt/dd标签

单元格合并
步骤：

  1. 明确合并哪几个单元格
  2. 通过左上原则，确定保留谁删除谁
    上下合并保留上，左右合并保留左
  3. 给保留的单元格设置格式
    属性名 属性值 说明
    rowspan 合并单元格个数 跨行合并（上下合并）
    colspan 合并单元格个数 跨列合并（左右合并）

注意：只有同一个结构标签中的单元格才能合并，不能跨结构标签合并（不能跨thead、tbody、tfoot合并）

属性加在td标签中，例如：<\td rowspan=’2’>

input系列

这个标签可以通过type属性值的不同，展示不同的效果

type属性值 说明

file 文件选择，用于之后上传文件

button 普通按钮，默认无功能，之后配合js添加功能
placeholder 占位符，提示用户输入内容的文本

单选框

属性名 说明
name 分组，有相同name属性值的为一组，一组中同时只能有一个被选中
cheaked 默认选中

注意：name属性对于单选框有分组功能；有相同name属性值的单选框为一组，一组中只能有一个被选中

文件选择

属性名 说明
mutiple 多文件选择

按钮

标签名 属性
submit 提交按钮
reset 重置按钮
button 普通按钮，默认无功能，之后配合js添加功能
要实现以上按钮功能，需要配合form标签使用，即用form标签把表单标签包含起来即可。

注：submit和reset要有表单域才能用，普通按钮可以添加属性value=”按钮说明”

button按钮标签

用于在网页中显示用户点击的按钮，是双标签.
标签名为button的属性：

type属性值 说明
submit 提交按钮，点击之后提交
reset 重置按钮
button 普通按钮，默认无功能，可用js加功能

select下拉菜单标签

用于提供多个选择的下拉菜单表单控件
标签组成：
select标签：下拉菜单的整体
option标签：下拉菜单的每一项
selected：下拉菜单的默认选中

textarea文本域标签

在网页中提供可输入多行文本的表单空间
标签名：textarea
属性：

  • cols:规定了文本域内可见宽度
  • rows:规定了文本域内可见行数

注意点：

  • 右下角可以拖拽改变大小
  • 实际开发时针对样式效果推荐使用css设置

label标签

场景：常用于绑定内容与表单标签的关系
使用方法1：

  • 用label标签把内容（如：文本）包裹起来
  • 在表单中添加id属性
  • 在label标签的for属性中设置对应的id属性 使用方法2：
  • 直接用label标签把内容和表单标签一起包裹起来
  • 需要把label标签的for属性删除即可

语义化标签

没有语义的布局标签div和span
场景：实际开发网页时会大量频繁地使用div和span

  • div标签：一行只显示一个（独占一行）
  • span标签：一行可以显示多个

有语义的标签
场景：在html5中，推出了一些给手机端用户使用

标签名 语义
header 网页头部
nav 网页导航
footer 网页底部
aside 网页侧边栏
section 网页区块
article 网页文章

注：以上显示特点和div一样，但是比div多了不同的属性

字符实体

场景：在网页中展示特殊符号时，需要使用字符实体替代
结构：&英文
常见字符实体：空格 &nbsp分号

设置滚动字幕

1 .默认情况下，滚动文字从右向做滚动
2.属性：direction可以修改字幕滚动方向
属性值：up 从下往上
down 从上往下
left 从右往左
right 从左往右

属性：behavior设置滚动字幕的滚动行为
属性值：scroll 设置文字循环滚动（默认）
slide 只进行一次滚动
alternate 循环交替往返滚动

属性：scorllamount 设置滚动速度 默认速度值为6
scrolldelay 设置滚动延迟 默认100毫秒
属性值：value，取值为数字
